## 📄 Description
Analog Clock is a simple yet elegant project that showcases an analog clock created using HTML, CSS, and JavaScript. The clock features light and dark modes, which are stored in local storage, allowing users to switch between them seamlessly.

## 🌟 Features
* Analog clock face with hour, minute, and second hands
* Light and dark mode toggle
* Modes are stored in local storage for a personalized experience
* Smooth and responsive design

## 🛠️ Tech Stack
* HTML: Structure and content
* CSS: Styling and layout
* JavaScript: Dynamic functionality and event handling

## 🗂️ Project Structure
* `index.html`: The main entry point of the application
* `script.js`: Contains the JavaScript code for dynamic functionality
* `style.css`: Styles and layouts for the application
* `README.md`: This file, containing information about the project

## 📦 Installation
1. Clone the repository using `git clone https://github.com/kiranShamsHere/analogClock.git`
2. Open `index.html` in a web browser to view the analog clock

## 🕒 Usage
1. Open `index.html` in a web browser
2. Toggle between light and dark modes by clicking on the mode switcher
3. Observe the smooth and responsive design of the analog clock
